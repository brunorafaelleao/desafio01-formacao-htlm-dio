O que são tags?

Índice

As tags HTML servem para estruturar o conteúdo de uma página da web. Elas ajudam a organizar o conteúdo em uma estrutura hierárquica e facilitam a navegação.

Tags de título ou cabeçalhos

Índice

As tags de título ou cabeçalhos são usadas para definir títulos e subtítulos em uma página da web. Elas vão de <h1> a <h6>, sendo <h1> o título mais importante e <h6> o menos importante.

Tags de páragrafo

Índice

A tag de páragrafo <p> é usada para definir um bloco de texto em uma página da web. Ela ajuda a separar o texto em parágrafos distintos, tornando a leitura mais fácil e organizada.

Tags de lista não ordenada

Índice

A tag de lista não ordenada <ul> é usada para criar listas de itens onde a ordem dos itens não é importante. Cada item da lista é definido pela tag <li>.

Tags de lista ordenada

Índice

A tag de lista ordenada <ol> é usada para criar listas de itens onde a ordem dos itens é importante. Cada item da lista é definido pela tag <li>.

Tags de formatação/personalização de textos

Índice

Existem várias tags HTML que permitem formatar e personalizar o texto em uma página da web. Aqui estão algumas das mais comuns:

a) Tag de itálico

A tag <i> é usada para deixar o texto em itálico. Exemplo: Texto em itálico

b) Tag de negrito

A tag <b> é usada para deixar o texto em negrito. Exemplo: Texto em negrito

c) Tag de sublinhado

A tag <u> é usada para sublinhar o texto. Exemplo: Texto sublinhado

d) Tags de sobrescrito e subescrito

A tag <sup> é usada para sobrescrever o texto, enquanto a tag <sub> é usada para subescrever o texto. Exemplo: H2O e E = mc2
